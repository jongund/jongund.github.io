Voice Over Screen Reader Concepts and Selected Commands

Keyboard Command Description
Command+F5 Start/Stop Voice Over on MacOS
Control+Option Voice Over (VO) command
Control Stop speaking
Control+Option++ Volume
Control+Option+A Starting reading.
Control+Option+P read next paragraph.
Control+Option+S read next sentence.
Control+Option+Right Arrow Next item on web page.
Control+Option+Left Arrow Previous item on web page.
Control+Option+U Open rotor to navigate content on the web page and use the left or right arrow keys to show the list of headings.
Control+Option+Command+H Next heading
Control+Option+Shift+Command+H Previous heading
Control+Option+F2 Read page title

Feature Overview

  • Voice Over allows you to view a list of headings.
  • When navigating a web page Voice Over announces heading information when the browse or focus cursor enters a new heading.
