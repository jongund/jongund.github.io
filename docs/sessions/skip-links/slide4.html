Traditional Skip links

  • Example code for Traditional Skip Link
  • Use <a href="#main">Skip to main content</a> landmark as target for link.
  • Use <main id="main">....</main> landmark as target for link.
  • Javascript focus and blur event handlers are used to make the link visible on focus and hidden when it does not have focus.
